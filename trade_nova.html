3. How to Buy NOVA

Purchase Method: `buy`

Functionality:

  • Use this method to buy NOVA with SOL.
  • The method automatically calculates and distributes additional NOVA rewards based on the current transaction phase.
  • A 5% fee is applied to each purchase, which supports the developer reward pool and liquidity pool maintenance.

Parameters:

  • `sol_amount`: The amount of SOL you wish to spend (minimum of 0.0001 SOL).
  • `current_nova_price`: The current price per NOVA token, typically provided by the contract or user interface.

Execution Process:

  1. Transfer SOL to Liquidity Pool PDA:
    • Send the specified amount of SOL from your wallet to the contract's Liquidity Pool PDA address.
  2. Receive NOVA Tokens:
    • Based on the current NOVA price and reward percentage, receive the corresponding amount of NOVA tokens in your wallet.
  3. Fee Deduction:
    • A 5% fee is deducted from your purchase amount and allocated to the Developer Reward Pool to support ongoing development and liquidity.

Example Usage:

Below is an example of how to use the buy function using Solana's Web3.js library in a JavaScript environment: 


// Import necessary libraries
const { Connection, PublicKey, clusterApiUrl, Keypair, Transaction, SystemProgram } = require('@solana/web3.js');

// Initialize connection
const connection = new Connection(clusterApiUrl('mainnet-beta'), 'confirmed');

// Define the contract address (replace with actual contract PDA)
const contractAddress = new PublicKey('7UJH6UbAcMANjoqBvmncVq7UK6jX6qQawDKqWDa4Vj3q');

// Buyer wallet keypair (ensure this is securely managed)
const buyerKeypair = Keypair.fromSecretKey(Uint8Array.from([/* Your Secret Key Here */]));

// Function to buy NOVA
async function buyNOVA(solAmount, currentNOVAPrice) {
    const transaction = new Transaction();

    // Transfer SOL to the contract's Liquidity Pool PDA
    transaction.add(
        SystemProgram.transfer({
            fromPubkey: buyerKeypair.publicKey,
            toPubkey: contractAddress,
            lamports: solAmount * 1e9, // Convert SOL to lamports
        })
    );

    // Sign and send the transaction
    const signature = await connection.sendTransaction(transaction, [buyerKeypair]);
    await connection.confirmTransaction(signature, 'confirmed');

    console.log('Purchase transaction successful with signature:', signature);
}

// Example purchase of 1 SOL
buyNOVA(1, 0.05); // Replace 0.05 with the actual current NOVA price

  • `amount`: The amount of NOVA tokens to transfer.

Execution Process:

  1. Verify Balance:
    • Ensure the sender has sufficient NOVA balance to cover the transfer amount.
  2. Execute Transfer:
    • Initiate the transfer of the specified NOVA amount from the sender's account to the recipient's account.
